# Login System Java

Sistema de login desenvolvido em Java com interface gráfica utilizando Swing.

## 🚀 Funcionalidades

- Validação de usuário e senha
- Armazenamento de dados em arquivo `.txt`
- Interface gráfica interativa
- Feedback de erro e sucesso

## 🛠️ Tecnologias

- Java
- Java Swing

## ▶️ Como executar

1. Clone o repositório
2. Abra na sua IDE (NetBeans)
3. Execute a classe principal

## 📚 Aprendizados

- Manipulação de dados de usuário
- Validação de login e senha
- Criação de interfaces gráficas com Swing
- Leitura e escrita em arquivos `.txt`
